Docker allows you to run DynaRetire locally on your own computer without needing to access our website. This guide will walk you through installing Docker and setting up DynaRetire on your personal machine.

What you'll get: A private copy of DynaRetire running on your computer that works even without internet access and keeps your financial data completely private.

Part 1: Installing Docker on Your Computer

For Windows Users:

  • Check your Windows version

    Make sure you have Windows 10 or newer.

  • Download Docker Desktop
    • Visit Docker's official download page
    • Click the "Download for Windows" button
    • Wait for the installer file to download
  • Install Docker Desktop
    • Double-click the downloaded file and follow the simple installation instructions
    • Restart your computer when prompted
    • After restarting, Docker will start automatically (look for the whale icon in your taskbar)

For Mac Users:

  • Check your Mac

    Make sure you have macOS version 10.15 (Catalina) or newer.

  • Download Docker Desktop
    • Visit Docker's official download page
    • Click the "Download for Mac" button
    • Wait for the installer file to download
  • Install Docker Desktop
    • Open the downloaded .dmg file
    • Drag the Docker icon to your Applications folder
    • Open Docker from your Applications folder

For Linux Users:

Linux installation varies by distribution. Visit the official Docker installation guide for Linux and select your distribution for specific instructions.

Part 2: Running DynaRetire Using Our Simple Script

We've created an easy-to-use script that handles all the technical details for you. Here's how to use it:

Step 1: Download the Deployment Script

  • Download our deploy_script.sh file
  • Save it somewhere easy to find, like your Downloads folder or Desktop

Step 2: Set Up Your Environment

You'll need to provide two pieces of information to access DynaRetire:

  • Your GitHub username
  • A personal access token (like a special password for GitHub)

For Windows users:

Open Command Prompt and type:

set DOCKER_USER=your_github_username
set DOCKER_TOKEN=your_personal_access_token
set DOCKER_PORT=5000

For Mac and Linux users:

Open Terminal and type:

export DOCKER_USER=your_github_username
export DOCKER_TOKEN=your_personal_access_token
export DOCKER_PORT=5000
Note: Replace the values with your personal information:
  • your_github_username: Your GitHub username
  • your_personal_access_token: Your GitHub personal access token
  • 5000: The port number (can be changed if needed)
What these variables mean:
  • DOCKER_USER: Your GitHub username for authentication
  • DOCKER_TOKEN: Your GitHub personal access token (serves as a password)
  • DOCKER_PORT: The port number where DynaRetire will be available (default: 5000)

Note: Replace "your_github_username" with your actual GitHub username and "your_github_token" with your personal access token. If you want DynaRetire to use a different port than 5000, you can change that number.

Step 3: Run the Script

For Windows users:

In the same Command Prompt window, navigate to where you saved the script and run it:

cd C:\path\to\folder
bash deploy_script.sh

For Mac and Linux users:

In the same Terminal window, navigate to where you saved the script and run it:

cd /path/to/folder
chmod +x deploy_script.sh
./deploy_script.sh

Step 4: Access DynaRetire

  • Once the script completes successfully, open your web browser
  • Go to: http://localhost:5000 (or the port you specified)
  • You should now see DynaRetire running locally on your computer!

Privacy Benefit: When running locally, your financial data never leaves your computer.

Checking Logs (If Something Goes Wrong)

If you encounter issues, you can check the log files to see what happened:

  • Look in the folder: /var/log/dynaretire/ (or the folder you specified with LOG_DIR environment variable)
  • You can view the logs with any text editor or using these commands:

For Windows users (PowerShell):

type C:\var\log\dynaretire\app.log

For Mac and Linux users:

tail -f /var/log/dynaretire/app.log    # View live application logs
less /var/log/dynaretire/error.log     # Browse error logs

Stopping and Starting DynaRetire

When you're done using DynaRetire:

For all users:

docker stop dynaretire

To restart it later:

docker start dynaretire

Troubleshooting Common Issues

If Docker won't start:

  • Restart your computer and try again
  • Make sure your computer meets the minimum requirements for Docker
  • On Windows, you might need to enable virtualization in your BIOS settings

If you see "Authentication Error":

  • Double-check your GitHub username and token
  • Make sure you typed them correctly in the environment variables
  • If you're not sure about your token, you can create a new one on the GitHub website

If you can't access DynaRetire at localhost:5000:

  • Make sure Docker is running (look for the whale icon)
  • Check if the script finished successfully without error messages
  • Try using a different port by setting DOCKER_PORT to another number (like 8080)
  • Check if another program is already using port 5000 on your computer
